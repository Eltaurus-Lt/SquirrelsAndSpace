pLTlib

A lightweight engine for making interactive mathematical graphics with a focus on performance and openness to easy low-level customization.



Getting started

Plot template

A typical code for a plot would look like the following: <div class="plot" PlotRange="[-5, 5]">
 <svg class="layer" id="plotCanvas"></svg>
 <div class="layer" Axes Arrowheads Ticks="1"></div>
</div> A plot can contain any number and various types of layers. The layers are rendered in reverse order: the one that comes first is placed at the bottom, the next is placed on top of it, etc. To draw the axes below the plot lines, for example, the <div> with Axes should be moved before the <svg>.

Structural plot properties are set up as html attributes of the .plot container and individual .layer elements. The appearance of any component can be customized using standard CSS.

Graphical content can be added to layers using the provided plotting methods and combined with static elements hardcoded directly inside the respective layers, e.g.: <svg class="layer" id="plotCanvas">
 <circle r="0.5" cx="2" cy="1" ></circle>
</svg>

For adding plot controls and making the elements interactive see the section on dynamic evaluation.

Boilerplate

Individual illustrations are intended to be coded in separate .html files (each can contain multiple .plot objects), which then are embedded into a common article page.

Import

The code to be included in the <head> of a plot html file: <script defer src="https://cdn.jsdelivr.net/gh/Eltaurus-Lt/SquirrelsAndSpace/pLTlib/pLTlib.js"></script>
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/Eltaurus-Lt/SquirrelsAndSpace/pLTlib/pLTlib.css">
<script defer src="https://cdn.jsdelivr.net/npm/mathjax@4/tex-mml-chtml.js"></script> The last line is for importing MathJax and can be ommited if LaTeX fonts and formatting are not used.

Embedding

The code for embedding into a parent document: <iframe src="plot.html" width="600px" onload="window.addEventListener('message', e=>{if(e.source===this.contentWindow && e.data && e.data.type==='embed-height'){this.style.height=e.data.height+'px';}});"></iframe>

Replace plot.html with the actual path to the plot file being embedded. The width attribute can be modified as needed and additional attributes, like classes and styling, can be inserted as usual. The included onload function ensures the height of the embedded plot is fit to the proper size of the source html and accurately reacts to page resizing.
